# 🎮 LoL Win Rate Tracker

## 🇬🇧 English

### 📌 Description

LoL Win Rate Tracker is a simple Windows Forms application that helps you track your wins, losses, and overall win rate across different game modes (e.g., SoloQ, Flex, Normal).

---

### 🚀 How to Run the Application

#### 🔹 Option 1: Run from Source Code (Recommended for Developers)

1. **Install prerequisites**

   * Windows OS
   * Visual Studio 2019 or 2022
   * .NET Framework 4.7.2 (or newer)

2. **Clone the repository**

   ```bash
   git clone https://github.com/dimitris01n/LoLWinrateTracker.git
   ```

3. **Open the project**

   * Open `LolApps.sln` with Visual Studio

4. **Restore dependencies**

   * Visual Studio will automatically restore NuGet packages
     (or go to: `Tools → NuGet Package Manager → Restore`)

5. **Run the application**

   * Press `F5` or click **Start**

---

#### 🔹 Option 2: Run Prebuilt Version (if provided)

1. Go to the **Releases** section
2. Download the `.zip` or installer
3. Extract the files
4. Run:

   ```text
   WinRateTracker.exe
   ```
